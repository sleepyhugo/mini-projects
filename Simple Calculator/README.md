# CLI Calculator

A simple command-line calculator written in C++ that supports basic arithmetic operations.

## Features
- Addition, subtraction, multiplication, division
- Input validation
- Division-by-zero protection
- Looping execution until user quits

## Concepts Used
- Functions
- Loops
- Conditional logic
- User input/output

## How to Run
```
g++ main.cpp -o calculator
./calculator
```
